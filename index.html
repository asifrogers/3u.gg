Features

Case-Insensitive

3u.gg short URLs are case-insensitive optimized for non-clickable media advertising or typing by hand. Read More

3u.gg/McDonalds is the same as 3u.gg/MCDONALDS or 3u.gg/mcdonalds. They all point to the same target URLs.

Embed Query Parameters for Campaign Tracking

3u.gg short URLs can embed campaign tracking parameters.

Such parameters (usually called UTM parameters) may be added to the target URLs or kept stored on the 3u.gg forwarding servers.

Editable

3u.gg short URLs are premium URLs. Owners can reuse them time and again by changing their target URLs and/or by changing their embedded campaign tracking parameters.

Choice URLs (if available)

3u.gg lets you choose the ending (if available).

You can grab your brand name (e.g. 3u.gg/McDonalds) or can choose other single dictionary words or their combinations.

3u.gg FAQs

Can a short URL be reused?
Yes. A 3u.gg short URL purchased is your long term asset. You can change the target URL as many times as you want.
Can I test the service before paying?
You can register your preferred short URL, provided it is available, without any upfront payment. The URL will be held in your account for a specified "hold time," during which you can test it. The hold time is 24 hours for four-character short URLs and one week for URLs with five or more characters.
What type of target URLs are supported?
Any URL that starts with a http or https is supported. The URL can point to HTML web pages, PDF documents and even dynamically generated content with query params like ?area=ny.
Are 3u.gg short URLs case-sensitive?
While the URL path as per official http protocol is case-sensitive, all 3u.gg short URLs are not. This is to make it easy for someone to type from memory. Read More
What types of payments do you support?
For now, we only support PayPal.
Do you support short URL creating by anonymous account?
No. Users need to sign in through SSO (Single Sign-on) through some other provider like Google.
Can 3u.gg short URLs be sold in a secondary market?
We don't endorse this practice, but you are free to sell your valuable 3u.gg short URLs on any secondary market. Transfers to another party are only permitted after payment is completed. This policy is designed to prevent squatting and URL kiting.
Do you preserve the original URL referrer while forwarding?
Yes. The URL referrer is forwarded in an http attribute.
What is the best practice to not trigger behaviour tracking blockers on my short URL?
Do not put any query params while putting short URLs on webpages. Embed all query params while creating the short URL. Read blog.
What is a typical response speed?
Depending on the geographical location of the client. It is usually very fast.
If I change the target URL, how long does it take to propagate?
In our system, the changes propagate almost instantaneously. The changes also propagate quickly through the DNS as we use the non-caching response http 302 instead of usual http 301.
Are there any dependencies on the browser type of version?
No. 3u.gg uses standard HTTP response codes supported by all browsers.
Do you have an API that we can use for further automation?
Yes. The API access is only granted upon request. Please contact us on support@3u.gg
Can I point target URLs with dynamic content?
Yes. You can. But this practice may trigger behaviour tracking URL blockers.
If I change targets, how are the usage statistics reported?
Usage statistics are reported separately for each target URL change.
What characters can be used in a short URL?
Any characters supported by HTTP protocol i.e. a-z, A-Z, 0-9. Other characters have special meanings.
What is the purpose of an Organization Account?
This is to give all your short URLs the same smaller folder prefix or a branded starting subdomain.
What is the minimum length of an Organization URL prefix?
Branded URLs start with a single character in your branded space. For a prefix based organization, it looks like 3u.gg/pr/_ _ _ _ and for branded organization it looks like pr.3u.gg/_ _ _ _
I want to report abuse from some user of 3u.gg service.
Please email us on abuse@3u.gg
Are there any restrictions on URL targets?
Yes. Phishing, porn and other targets are banned. Please refer to our Terms of Service.
What is the origin of .gg TLD?
.gg is the official TLD of Bailiwick of Guernsey, part of British Channel Islands and follows the British Law.
Is my data safe?
Yes. Please refer to our Privacy Policy.
